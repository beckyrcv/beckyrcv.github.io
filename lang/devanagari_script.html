Typing Hindi (Devanagari)

There are several ways to type Devanagari (the script used for writing Hindi) on modern computers:

  • You can copy/paste from a web dictionary or translation service (not recommended - can't edit! but will work in a pinch).
  • You can use a web keyboard to either click provided buttons or type on directly your keyboard. Text in Devanagari will appear in the webpage textbox. For example, Lexilogos has a nice set of web-based language keyboards. You can try out their Devanagari keyboard here.
  • One of the most powerful language entry methods is to use language input features from your operating system (i.e. built into your computer). I primarialy use this method.

Grammar Review

Verb types

खाना (verb form) -> मैं खाता हूँ (था) (present habitual (future!)) ->तम खा रही हो (present progressive) -> आप खाएे (imperative) -> मैं हूँ (regular present - form of होना)

Verb Agreement

For subject + को: The verb agrees with either the object (if not oblique) or then the indirect object (if not oblique) or then uses the default agreement (3rd person singular masculine).
Note: any noun in oblique form will never be used for verb agreement!
